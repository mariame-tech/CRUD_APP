# Api de Myspace 

C est une api qui permets de gerer les crud sur les users et les messages entre users

# Technologies 
- nodejs
- typeorm
- mysql 
- linter 

# run le projet
 
1.  `  git clone https://github.com/darcia0001/myspace.git `
2.  `  cp myspace `
3.  `  npm i  `
4.  `   npm run dev   `
